Events
On Touched
Triggers when parent part is collided with.
On Clicked
Triggers when the object is tapped.
On Start
Triggers immediately when simulating begins.
On Update
Triggers repeatedly every frame.
Actions
Set Color
Modifies the target part's color.
Set Transparency
Sets the transparency of a part.
Set Anchored
Locks or unlocks physics simulation.
Set Size
Scales the target size factor.
Set Visible
Toggles target visual visibility.
Set CanCollide
Sets collidability status.
Launch Up
Launches the target upward.
Damage Humanoid
Inflicts damage to the target humanoid.
Heal Humanoid
Restores health to the target humanoid.
Teleport To
Teleports object to target position.
Set Gravity
Changes world gravity simulation.
Rotate Object
Rotates the target around the Y-axis.
Apply Impulse
Applies a physics impulse vector.
Move Relative
Moves the object relative to itself.
Set WalkSpeed
Sets walk speed of character humanoid.
Set JumpPower
Sets jump power of character humanoid.
Set Health
Sets target health directly.
Kill Target
Kills target humanoid immediately.
Destroy Object
Removes the target from workspace.
Point Light
Toggles point light inside target.
Set Sky Gradient
Dynamically transitions skybox colors.
